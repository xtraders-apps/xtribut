DESCRIÇÃO DO SOFTWARE
---------------------
O XTributation é um sistema web desenvolvido para auxiliar investidores 
brasileiros que operam no mercado americano de ações a calcular e gerenciar
suas obrigações tributárias. O software oferece funcionalidades de:

- Cálculo de variação cambial para envios e retiradas de recursos
- Apuração de Imposto de Renda sobre operações em bolsa estrangeira
- Geração de relatórios fiscais detalhados
- Histórico completo de operações e transações
- Exportação de dados para declaração de imposto de renda
